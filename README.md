## Requirements
* [node & npm](https://nodejs.org/en/download/)

## Installation
Using the command line, run the following commands inside the project's root folder:  
```bash
npm install
gem install jekyll
gem install i18n
```

Then run `jekyll build` to build the website, or `jekyll server` to serve it.
